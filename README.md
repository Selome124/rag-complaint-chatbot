# Intelligent Complaint Analysis Chatbot (RAG)

This project implements a Retrieval-Augmented Generation (RAG) chatbot for analyzing customer complaints in financial services.

## Business Context
CrediTrust Financial receives thousands of customer complaints monthly across multiple products. This system enables internal teams to query complaints using natural language and receive evidence-backed insights.
